# Ninja List 🥷

### Descrição do Projeto
Ninja List é um **projeto pessoal** desenvolvido em **React + Vite** com o objetivo de **praticar e aprimorar habilidades em JavaScript, React, e desenvolvimento de interfaces modernas**.  
O sistema é um **gerenciador de tarefas simples** com **CRUD completo** e uma interface visualmente atrativa e intuitiva.



### Tecnologias Utilizadas
- **React 18**
- **Vite** (bundler ultrarrápido)
- **CSS / Flexbox** para layout
- **React Router DOM** para navegação entre páginas
- **Axios** para integração com backend



### Funcionalidades
- **Adicionar tarefas** com título e descrição.
- **Editar tarefas existentes**.
- **Excluir tarefas**.
- **Marcar tarefas como concluídas ou pendentes**.
- **Filtrar tarefas por status**.
- **Interface responsiva e atraente**, com destaque visual para prioridade de tarefas.

## 1️⃣ Clonar o repositório

```bash
git clone https://github.com/ThiagoRicch/ninjalist-frontend.git
```

## 2️⃣ Entrar na pasta do projeto

```bash
cd ninja-list
```

## 3️⃣ Instalar dependências

```bash
npm install
```
ou

```bash
yarn install
```

## 4️⃣ Rodar a aplicação

```bash
npm run dev
```
ou

```bash
yarn dev
```

O projeto será aberto no navegador automaticamente, normalmente em http://localhost:5173
